About Project

Project Brief: The Ford GoBike system, now known as Bay Wheels, is a bike-sharing program that operates in the San Francisco Bay Area. This analysis aims to uncover usage patterns, peak usage times, and user demographics to better understand how the system is utilized and identify areas for improvement.

Project Objective: The primary goal of this analysis is to explore the data from the Ford GoBike system to derive insights that can help improve service operations, enhance user experience, and support decision-making for system expansion and resource allocation.

Data Collection: This data set includes information about individual rides made in a bike-sharing system covering the greater San Francisco Bay area. Here is a link to the dataset, The Data consists of 13 variables and includes:

  • Trip Duration (seconds)
  • Start Time and Date
  • End Time and Date
  • Start Station ID
  • Start Station Name
  • Start Station Name
  • Start Station Longitude
  • End Station ID
  • End Station Name
  • End Station Latitude
  • End Station Longitude
  • Bike ID
  • User Type (Subscriber or Customer – “Subscriber” = Member or “Customer” = Casual)

    • Some of the data wrangling steps carried out during the analysis involves and not limited to change of dtype of some columns, creating a new column from previous one, and removing irrelevant columns as well as rows with missing entries.

    Summary of Findings: The bike-sharing service in San Francisco is primarily used for short commutes or errands, with peak usage during morning and evening rush hours. Key stations in downtown San Francisco and near major transit hubs are popular, while some are underutilized. The user base is predominantly male, aged 25-34, with weekday usage higher than weekends. Seasonal trends show higher usage during warmer months and a dip in winter.

    Conclusions: The Ford GoBike system data reveals distinct patterns in user behavior, station usage, and temporal trends. The casual users travel twice more than their clients in average duration of trips in seconds. Ages between 25 and 40 are more interested in Riding Bike

    Recommendations:

  • Increase bike and dock availability at high-demand stations to reduce congestion and improve user satisfaction.
  • Develop marketing campaigns aimed at attracting a more diverse user base, particularly focusing on underrepresented groups.
  • Introduce seasonal promotions or alternative services during winter months to maintain steady ridership.
